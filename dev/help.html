Fitch Web

Fitch Web is a web-based proof notating and verifying program for writing natural deduction proofs in propositional logic.

How to Use

Below an example of a proof of ⊢(((A ∧ B) → C) ↔ (A → (B → C))) is provided as a guide for syntax. See below for a table of operators (WIP). 

|_
| |_(A&B)->C
| | |_A
| | | |_B
| | | | A
| | | | A&B
| | | | (A&B)->C
| | | | C
| | | B->C
| | A->(B->C)
| |_(A->(B->C))
| | |_A&B
| | | A
| | | A->(B->C)
| | | B->C
| | | B
| | | C
| | (A&B)->C
| ((A&B)->C)<->(A->(B->C))

(Note that your proof must end in a newline.)

Notice that after putting the example into the input box (including the newline at the end), a (slightly) formatted version of the proof will appear to the right or below of the input box.

To the right of this proof will also be a column of text inputs. These inputs are used to verify the proof; each input being a justification of its corresponding proof line.

A justification is made up of a rule, then some lines that fulfill the requirements of the rule. The justifications for the example are provided below, starting with an Assumption corresponding to the empty assumption at the top.
